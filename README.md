## 系統簡介

我們為了解決逢甲大學周邊因停車空間不足，導致學生面臨違停拖吊風險與通勤焦慮，以及大量車輛繞行尋車所造成的商圈交通壅塞與人車爭道等痛點，規劃了「校園停車way查詢系統」。首先透過 AI 協作擴充需求，隨後經小組務實評估，考量硬體與資料庫串接門檻，捨棄了車牌辨識與校務系統綁定等高難度功能。最終將核心聚焦於開發「實時停車與特殊車位查詢」及「多模式智慧導航」，並確立了系統需具備即時性、高可用性與資安防護等非功能性需求，產出一份兼顧解決痛點與技術可行性的軟體需求藍圖。

## 個人心得

張嘉拓
解決真實的問題很有成就感 這不是一個為了做而做的功課，而是真的能解決學校周邊塞車、找車位痛苦的問題。想到這個系統還能順便引導人潮去逛旁邊的商店街，就覺得寫出去的東西是有溫度的、能實際幫助到大家。先想清楚再動手，真的省很多時間 我們沒有一開始就猛敲鍵盤寫程式，而是先乖乖把「要做什麼功能」、「畫面怎麼連」都畫下來、想清楚。本來以為這樣很花時間，結果寫程式的時候反而超順，根本不用一直回頭改來改去！跟 AI 合作就像多了一個超強神隊友 這次用 AI 來幫忙，感覺不只是「它幫我寫作業」，而是像在帶一個神隊友。我負責出主意、決定方向，AI 把那些繁瑣的程式碼跟容易出錯的小細節搞定。這讓我能把精力放在系統到底好不好用上面。先求有生出來，再求好 有時候很容易卡在「想把它做得很完美」，但這次我們把握一個原則：把最重要的「找車位」跟「繳費」先做出來就好！先把最核心的東西做出來跑跑看，真的很有成就感，之後想要加什麼酷炫的功能也都不遲。整個過程從無到有看著網站架起來，真的是一次很好玩又很有收穫的體驗！
